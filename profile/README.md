# 🍽️ Bereket Döner | GitHub Organizasyonu

**Lezzetin Arkasındaki Yazılım Gücü**

Merhaba! Burası Bereket Döner'in resmi GitHub organizasyonudur. Türkiye'nin dört bir yanında hizmet veren lezzet markamızın dijital dünyadaki yansıması burada hayat buluyor. 🧑‍💻

## 📌 Hakkımızda

Bereket Döner olarak sadece döner üretmekle kalmıyor, aynı zamanda tüm operasyonel süreçlerimizi dijitalleştirmek amacıyla **kendi kurum içi (in-house) kullanımımıza özel** yazılım çözümleri geliştiriyoruz.


## 💻 Teknolojilerimiz

Projelerimizde kullandığımız başlıca teknolojiler:

- `C# / .NET Framework / .NET Core`
- `MS SQL Server`
- `TypeScript / JavaScript`
- `HTML / CSS / Bootstrap`
- `Serenity Framework`
- `Logo ERP Entegrasyonları`
- `RESTful API'ler`
- `Windows Services & Console Apps`

## 🎯 Hedefimiz

Yazılım ekibi olarak amacımız:
- Tüm operasyonel süreçlerimizi dijitalleştirmek
- Sürekli iyileştirme ve otomasyon kültürünü benimsemek

## 📂 Projelerimiz

Bu organizasyonda yer alan projelerin büyük bölümü **şirket içi (in-house)** ihtiyaçlara özel olarak geliştirilmektedir.
